# Стек

- **Чистый HTML + CSS**, без фреймворков и сборщиков. **Не используй Tailwind.**
- Стили пишутся во внутреннем `<style>` внутри каждого HTML-файла. Классы — свои, семантические (`shell`, `topbar`, `topnav`, `avail`…). Придерживайся уже существующих имён и стиля.
- Язык контента — **русский** (`<html lang="ru">`).
- Шрифты **локальные**, лежат в `fonts/` (Hanken Grotesk `.woff2`), плюс Unbounded с Google Fonts. Не заменяй их без просьбы.
- Картинки — **реальные файлы из `images/`**. Не подставляй плейсхолдеры (placehold.co и т.п.).
