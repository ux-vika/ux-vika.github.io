# Структура файлов

- `index.html` — главная страница портфолио.
- `index.backup.html` — резервная копия, **не трогать**.
- Кейсы — в подпапках, у каждого свой `index.html`: `antivirus/`, `badminton/` (есть ещё `proto.html`), `calendar/`, `mail/`, `proto/`, `recurring/`, `wallet/`.
- `cv.pdf` — резюме.
- `output/playwright/` — сюда пишутся скриншоты.
- `fonts/` — локальные шрифты, `images/` — картинки сайта.
